ğŸ“Œ Project Summary

The static analysis examines the deformation and stress response of the prism under a 2013 N transverse remote force, applied at one-third of its length from the fixed end. 

The transient analysis investigates the time-varying structural response under multiple applied forces on different faces, evaluating minimum, maximum, and average stresses and deformations across the simulation time.

A mesh convergence study was conducted to ensure numerical accuracy for both.
